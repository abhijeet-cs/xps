## 🧠 SQL Execution Order

- `FROM / JOIN` → choose tables and combine data  
- `WHERE` → remove unwanted rows early  
- `GROUP BY` → create groups  
- `HAVING` → filter groups  
- `SELECT` → choose columns / calculations  
- `DISTINCT` → remove duplicates  
- `ORDER BY` → sort final rows  
- `LIMIT / OFFSET` → restrict output rows
