## ЁЯза SQL Execution Order Mnemonic

- **F** = `FROM / JOIN` тЖТ choose tables and combine data  
- **W** = `WHERE` тЖТ remove unwanted rows early  
- **G** = `GROUP BY` тЖТ create groups  
- **H** = `HAVING` тЖТ filter groups  
- **S** = `SELECT` тЖТ choose columns / calculations  
- **D** = `DISTINCT` тЖТ remove duplicates  
- **O** = `ORDER BY` тЖТ sort final rows  
- **L** = `LIMIT / OFFSET` тЖТ restrict output rows
